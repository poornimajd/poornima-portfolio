Confidence Scoring to Detect Hallucinations in LLMs

Implemented the "LLMs Know More Than They Show" framework on the MedQuad dataset to detect hallucinations in large language models. Fine-tuned LLaMA 3.2 using QLoRA and utilized Mistral 7B for in-context learning to extract exact answer tokens. Achieved a 78% error detection accuracy using a logistic regression probing classifier.

GitHub: LLMs Know More Than They Show
